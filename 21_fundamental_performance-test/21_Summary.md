### Fundamental Performance Test
>Poin Poin yang dipelajari
1. Performance Test
2. Jenis Test
3. Understanding

>Performance Test
```
Teknik non functional testing untuk menentukan parameter sistem dalam hal responsif dan stabilitas dibawah berbagai beban kerja.

Performance testing mengukur kualitas atribut dari sistem seperti stabilitas, ketahanan, dan sumber daya.

Yang diukur:
- Performa
- Bukan functional
- Reliability
- Stability
- Availibility
- Response
- Interaksi

yang perlu diperhatikan:
- Throughput dan Response data

Metode:
- Membuat Test Plan
- Menentukan metode
- Beberapa tipenya (Load Test, Endurance Test, Stress Test, Peak Test)
```

>Jenis Test
```
Smoke Testing
>> Untuk Verify script yang dibuat apakah dapat dihandle sistem

Load Testing
>> Pengujian paling sederhana dengan memahami sistem pada kondisi tertentu. Mengetahui jumlah traffic sebagai nilai load test

Tabel Tier
>> Menjadi patokan ideal suatu sistem baru

Stress Testing
>> Merode untuk mengamati kemampuan dan kestabilan sistem pada kondisi ekstrem

Spike Testing
>> Seperti stress test namu kenaikan langsung menuju puncak dalam waktu singkat

Soak Test
>> Mengetahui reliability ketika dalam tekanan
```

>Understanding
```
Volume test adalah sebuah test non functional yang menguji ketahanan suatu sistem ketika iberi database yang banyak atau sebuah flood test, berbeda dengan load test yang menguji secara umum, volume test berfokus pada sistem databasenya.
```
