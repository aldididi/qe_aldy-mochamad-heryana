### WEB PROGRAMMING
>Poin Poin yang dipelajari
1. FRONT END dan BACK END
2. HTML
3. CSS

>Front End dan Back End
```
Front End adalah sebuah bagian dari website yang membuat tampilan user yang menarik

Back End adalah kebalikan dari Front End. Bagian ini mengelola belakang layar sebuah website atau aplikasi berbasis web, karena itu Back End sering juga disebut sebagai server-side. Orang yang berperan mengembangkan sisi ini adalah Back End Developer.
```

>HTML
```
HTML atau Hypertext Markup Language
Sebuah standar yang digunakan secara luas untuk menampilkan halaman web

Kegunaan HTML
- Membuat Struktur halaman website
- Mengatur Tampilan dan isi dari halaman web
- Membuat tabel dengan tag HTML table
- Membuat form HTML
- Membuat gambar dengan canvas.
- Mempublikasikan halaman website secara online.


```

>CSS
```
Cascading Style Sheet (CSS) merupakan aturan untuk mengatur beberapa komponen dalam sebuah web sehingga akan lebih terstruktur dan seragam. CSS bukan merupakan bahasa pemograman.

CSS dapat digunakan untuk mengatur desain, layout, dan variasi display untuk ditampilkan pada berbagai macam perangkat, ukuran layar, dan printer.

Fungsi CSS
1. Mempercepat Desain
2. Membuat tampilan lebih variatif
3. Membuat halaman lebih cepat dimuat
4. Mempermudah proses pemeliharaan
5. Membuat optimasi ke perangkat lebih mudah
```
